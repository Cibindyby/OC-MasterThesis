Folgende Notizen sind wichtig, wenn man die hpo mit Kommandozeilenparametern aufruft:

arg1: Datensatz
    0 -> Parity
    1 -> Encode
    2 -> Decode
    3 -> Multiply

arg2: Crossover Type
    0 -> single point
    1 -> two point
    2 -> uniform
    3 -> no crossover
    
arg3: Crossover Rate Type
    1 -> konstant
    2 -> Clegg et al. (fallend)
    3 -> one fifth rule

arg4: Offset
    0 -> ohne Offset
    1 -> mit Offset
